# Lagos Delivery Dash Web Game

Playable Nigerian-themed browser game where you ride through Lagos traffic, collect delivery packages, grab fuel, and avoid buses and potholes.

## Run

Open `index.html` in a browser.

## Controls

- Arrow keys or A/D to move between lanes
- Space to pause or resume
- Tap lane buttons on mobile

## Features

- canvas game loop
- lane-based movement
- package and fuel pickups
- bus and pothole hazards
- increasing level speed
- score, fuel, timer, pause, and high score
- localStorage high score
